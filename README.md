# NHGJ
задание в мудл
Шаг 1. Подготовка базы данных (если нет готовой)
1.Откройте SQL Server Management Studio (SSMS).
2.Выполните скрипт для создания базы и таблицы:
zШаг 2. Создание WPF-проекта в Visual Studio
1.Создайте новый проект типа "WPF Application" (.NET Framework или .NET Core/5+, но для .NET Core потребуется дополнительные пакеты EF Core).
2.Назовите проект, например, WpfDatabaseFirstDemo.
Шаг 3. Установка необходимых NuGet пакетов
1.Через меню "Проект" -> "Управление пакетами NuGet" или правой кнопкой по проекту.
2.Установите пакеты (для .NET Framework):
oEntityFramework
oSystem.Data.SqlClient (часто уже установлен)
Шаг 4. Добавление модели EDM (Database First)
1.Правой кнопкой мыши по проекту -> "Добавить" -> "Новый элемент".
2.В категории "Данные" выберите "ADO.NET Entity Data Model".
3.Назовите модель UniversityModel.edmx.
4.В мастере выберите "EF Designer from database" -> Далее.
5.Создайте новое подключение: укажите свой сервер и выберите базу University. Сохраните строку подключения в конфиге.
6.На следующем шаге выберите таблицу Students. Оставьте галки по умолчанию. Нажмите "Готово".
Шаг 5. Изучение сгенерированного кода
1.Откроется графический дизайнер с сущностью Student.
2.В "Обозревателе решений" разверните файл .edmx. Вы увидите:
oUniversityModel.tt -> Student.cs (класс-сущность).
oUniversityModel.Context.tt -> UniversityModel.Context.cs (класс контекста с DbSet<Student>).
